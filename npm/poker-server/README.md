# poker-server

Poker Planning server for agile teams.

## Installation

```bash
npm install -g poker-server
```

Or run directly with npx:

```bash
npx poker-server
```

## Usage

```bash
poker-server
```

The server listens on port `8888` by default.

## Features

- Real-time multiplayer poker planning sessions
- WebSocket-based communication for instant updates
- Vote privacy until the Scrum Master reveals

## Supported Platforms

- macOS (Apple Silicon / ARM64)
- Linux (x64)
- Windows (x64)
